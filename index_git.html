Definição do GIT

GIT é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase de velocidade. O GIT foi inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento de kernel Linux, mas foi adotado por muitos outros projetos.
